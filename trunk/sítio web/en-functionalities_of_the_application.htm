Functionalities of the Application

The developed aplication implements the following functionalities:

  • Drag & drop functionality to manipulate visual controls in the construction of interfaces;
  • Properties editor to format the visual controls implemented in the draw area;
  • Storage of visual interfaces in selected media type;
  • Storage of templates with the visual characteristics of formatted controls;
  • Application of templates of visual controls over controls of the same type;
  • Elimination of visual controls in the draw area;
  • Preview of the built interface in a browser;
  • Generation of HTML pages from the interfaces built.
